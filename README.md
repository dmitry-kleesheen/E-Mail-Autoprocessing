# E-Mail Autoprocessing

Automated e-mail processing and distribution system developed with VB.NET Windows Forms and SQLite.

## Overview

E-Mail Autoprocessing is a desktop application designed to automate the distribution of documents and files via e-mail. The system monitors specified server folders, matches files with recipient information stored in a SQLite database, and automatically sends e-mails through Microsoft Outlook.

The application helps reduce manual work and ensures timely delivery of documents to customers, partners, or internal departments.

## Features

* Automatic processing of files from server folders
* Recipient management using SQLite database
* E-mail distribution based on predefined rules
* Microsoft Outlook integration
* SMTP mail delivery support
* Logging and processing history
* Easy configuration through Windows Forms interface

## Technologies

* VB.NET
* Windows Forms
* SQLite
* Microsoft Outlook COM Automation
* SMTP
* File System Monitoring

## Business Benefits

* Eliminates repetitive manual e-mail sending
* Reduces processing time
* Improves document delivery reliability
* Centralized recipient management
* Suitable for corporate document workflows
